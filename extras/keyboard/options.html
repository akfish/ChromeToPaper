Instapaper Extras: Keyboard Shortcut

This extra allows you to set a Keyboard Shortcut to add a page to Instapaper.

Note: Not all keyboard shortcuts will work. Keyboard Shortcuts do not work on special protected pages where content scripts are prohibited. Changing your Keyboard Shortcut will not take effect on all pages until after they are reloaded

Click the button below to change the Keyboard Shortcut.
