In my effort to simplify the donation process, I've created different ways to donate. Please choose a method that will work best for you.

Paypal
Donate with Paypal.
Google One-Time
Purchase this Web App through the Google Chrome Webstore. You will be charged a one-time fee. Once you have donated, feel free to uninstall the Web App. Please note that this Web App has no useful features; it is used purely for donations.
Google Repeating
Purchase this Web App through the Google Chrome Webstore. As long as it is installed, you will automatically donate each month. Please note that this Web App has no useful features; it is used purely for donations.
