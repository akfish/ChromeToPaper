ChromeToPaper

Congratulations! You have the latest version of ChromeToPaper. Checkout the options page to configure this extension.

V2.5.4: Icons

You can now choose an icon via the options page! In addition to the "default" icon, there is currently a "green" icon (user-submitted) and the "classic" icon that I used while I was in the process of designing a new icon. Whatever icon you choose should take effect immediately. Please let me know if it doesn't.

There's currently no way to choose an icon from your own computer (due to permissions), but I am completely open to user-submitted icons. Complete details are available at the icon chooser page.

I'd like to thank the few loyal users who helped provide me with an Instapaper Subscription. OAuth has proven to be much more difficult than I expected, but it should be available in version 2.6 (maybe earlier).

V2.5.3: Bug Fix

A user was kind enough to let me know that both the "Text View on Save" and the "Close Tab on Save" features were broken in V2.5. The issue has now been fixed.

V2.5: Work in Progress

I've made another UI overhaul. It looks a lot nicer, and the code is much easier to manage. I'm still working on Icon Themes and a Browser Popup Menu.

To returning users, I'd like to thank you for continuing to install my extension. I know that the new permission requirement may seem creepy at first, but I've thought about it for some time, and that is the best way to integrate extensions together.

One of the immediate consequences of this is the nice bar at the top of the options page. By default, you will only have two links, Core and Manager. However, as you install other extensions in the family, they will magically appear there. At the moment, ChromeToPaper Core (this extension) is the only extension with full integration support.

As many of you know, there is a new API for Instapaper. This API uses OAuth, and comes with many new features, such as viewing your saved bookmarks. However, nearly all of these commands require an Instapaper subscription. At the moment, I can't afford an Instapaper Subscription. Once I can, I'll begin implementing it.
