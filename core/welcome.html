V2.3

Recently, I have received several feature requests for my extension that are either somewhat esoteric or downright controversial. Adding these features to my extension would discourage some users from using my extension, so I thought of a workaround.

With this new update, I have created a separate extension that can augment the features of this extension. I proudly present to you ChromeToPaper Keyboard Shortcut! As the name suggests, this extra will enable Keyboard Shortcuts integration with this extension.

V2.2:

The extension user interface has been modified to provide a more unified experience. Extension pages (like this one and the options page) should be more user friendly.

V2.1.1

Due to popular opinion, the Extension Updated Page (this page) no longer pops up immediately after an update. Instead, the icon displays a "msg" badge, which will open the Instapaper Updated Page on click.

V2.1

Version 2.0 was a huge upgrade which left plenty of room for improvement. One user, 2010 politely informed me that there was no feedback when the extension button is clicked until it is added. Another user, Andre, suggested that clicking the Instapaper button on the new tab page opened the instapaper site. I've accounted for both of these suggestions in this update.
