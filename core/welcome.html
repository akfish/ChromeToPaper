V2.5: Work in Progress

I've made another UI overhaul. It looks a lot nicer, and the code is much easier to manage. I'm still working on Icon Themes and a Browser Popup Menu.

To returning users, I'd like to thank you for continuing to install my extension. I know that the new permission requirement may seem creepy at first, but I've thought about it for some time, and that is the best way to integrate extensions together. One of the immediate consequences of this is the nice bar at the top of the options page. By default, you will only have two links, Core and Manager. However, as you install other extensions in the family, they will magically appear there. At the moment, ChromeToPaper Core (this extension) is the only extension with full integration support. As many of you know, there is a new API for Instapaper. This API uses OAuth, and comes with many new features, such as viewing your saved bookmarks. However, nearly all of these commands require an Instapaper subscription. At the moment, I can't afford an Instapaper Subscription. Once I can, I'll begin implementing it.
