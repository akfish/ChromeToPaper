Notice

A user has recently asked me to implement a keyboard shortcut to add a page to Instapaper. I have been avoiding the implementation of a keyboard shortcut for a while because of the way extension keyboard shortcuts currently work in Google Chrome. In order for an extension to detect keyboard shortcuts, it must inject a userscript into each website a user visits. This userscript would monitor the keyboard. In short, if I implement keyboard shortcuts, my extension will need to request for "Your data on all websites". I'm still debating whether or not I should implement shortcuts, but I just wanted to give you guys a heads up.
