V2.3

Recently, I have received several feature requests for my extension that are either somewhat esoteric or downright controversial. Adding these features to my extension would discourage some users from using my extension, so I thought of a workaround.

With this new update, support for Instapaper Extras, separate extensions that can augment the features of this extension, has been added. To start the party, I proudly present to you Instapaper Extras: Keyboard Shortcut! As the name suggests, this extra will enable Keyboard Shortcut integration with this extension. Please note that Instapaper Extras are an experimental feature and may contain severe bugs.

Last but not least, I am searching for a few users to serve as testers. Basically, you will be given BETA versions of my Instapaper extension (and its extras) to test out. Your only responsibilities would be to report bugs and offer suggestions. If you are interested, please check out this page.

V2.2:

The extension user interface has been modified to provide a more unified experience. Extension pages (like this one and the options page) should be more user friendly.

V2.1.1

Due to popular opinion, the Extension Updated Page (this page) no longer pops up immediately after an update. Instead, the Instapaper icon displays a "msg" badge, which will open the Instapaper Updated Page on click.

V2.1

Version 2.0 was a huge upgrade which left plenty of room for improvement. One user, 2010 politely informed me that there was no feedback when the extension button is clicked until it is added. Another user, Andre, suggested that clicking the Instapaper button on the new tab page opened the instapaper site. I've accounted for both of these suggestions in this update.

V2.0

With this breakthrough version, the extension interface has been completely recreated. The classic popup bubble has been replaced with Chrome-style extension badges. In addition, the options page has been heavily modified to accomodate the login.
